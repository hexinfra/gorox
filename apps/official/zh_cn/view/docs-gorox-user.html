用户手册

本文描述如何使用 Gorox。

如果您是一位开发者，在阅读完本文后，还希望学习如何使用 Gorox 进行 Web 应用或微服务开发，那么请继续参考 Gorox 开发者手册。

编译与安装

在编译 Gorox 前，请确保您的运行平台符合以下要求：

  • Linux 内核版本 >= 3.9，CPU 架构为 AMD64 或 ARM64；
  • FreeBSD >= 12.0，CPU 架构为 AMD64；
  • 苹果 macOS >= Catalina，CPU 架构为 AMD64 或 ARM64；
  • 微软 Windows >= 10，CPU 架构为 AMD64。

然后，请确保您安装了 Go >= 1.19 版本。

编译

shell> go build cmds/gomake.go
shell> ./gomake all

安装

在您编译成功，或者下载了二进制版本后，直接把 gorox 所在的目录移到您想要存放的位置，即可。
