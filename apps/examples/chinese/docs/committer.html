产品功能

Gorox 是一款务实的网络代理、Web 应用服务器和 RPC 框架。

由于 Gorox 高度灵活的架构设计，它具有非常强大的扩展能力，可以基于它开发各类服务。

功能体系

Router 和 Server

Gorox 的前端部分有两套子系统：Router 子系统和 Server 子系统。

Router 子系统涵盖的元素有：Router、Gate、Conn、Case、Dealet，用于实现 L4 代理以及非 Web 的 L7 代理。

Server 子系统包含通用体系、Web 体系和 RPC 体系，其中：

  • 通用体系涵盖的元素有：Server、Gate、Conn，用于开发者自行发挥实现各种通用服务器（例如 FTP 服务器、聊天服务器等）；
  • Web 体系涵盖的元素有：Server、Gate、Conn、Stream、Request、Response、Socket、Webapp、Rule、Handlet、Reviser、Socklet，用于实现 Web 服务器、Web 应用、Web 代理等；
  • RPC 体系涵盖的元素有：Server、Gate、Call、Req、Resp、Service、Bundlet，用于实现 RPC 服务。

Backend

Gorox 的后端部分为 Backend 子系统，包含通用体系、Web 体系和 RPC 体系，其中：

  • 通用体系涵盖的元素有：Backend、Node、Conn，主要由 Router 前端子系统使用；
  • Web 体系涵盖的元素有：Backend、Node、Conn、Stream、Request、Response、Socket，主要由 Web 前端体系使用；
  • RPC 体系涵盖的元素有：Backend、Node、Call、Req、Resp，主要由 RPC 前端体系使用。
